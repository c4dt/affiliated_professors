## Professors

### [Bryan Ford](professors/BRYAN-FORD.md)
*Decentralized and Distributed Systems Lab (DEDIS)* · last updated 2026-07-10

Bryan Ford leads the Decentralized and Distributed Systems (DEDIS) lab at EPFL, focusing on large-scale collective authorities ("cothorities"), scalable Byzantine consensus, privacy-preserving data sharing, and coercion-resistant e-voting. Recent highlights include **Limix** (NINeS 2026), a fault-isolation metadata service for globally-managed distributed systems, and **TRIP** (ACM CCS 2025), a coercion-resistant registration protocol for the Votegral e-voting platform. The lab actively maintains widely used open-source infrastructure including the **Kyber** cryptographic library and the **QuePaxa** consensus protocol, and explores foundational topics from blockchain rationality to grounded deduction in formal logic.
